# Check if an argument is a Nano ID string

Check if an argument is a Nano ID string

## Usage

``` r
is_nanoid(x)
```

## Arguments

- x:

  (`any`)  
  Object to check.

## Value

`TRUE` if `x` is a valid Nano ID string, `FALSE` otherwise.

## Examples

``` r
is_nanoid("V1StGXR8_Z5jdHi6B-myT")
#> [1] TRUE
```
