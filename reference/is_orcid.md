# Check if an argument is an ORCID string

Check if an argument is an ORCID string

## Usage

``` r
is_orcid(x)
```

## Arguments

- x:

  (`any`)  
  Object to check.

## Value

`TRUE` if `x` is a valid ORCID string, `FALSE` otherwise.
