# Check if an argument is a BIC/SWIFT code string

Check if an argument is a BIC/SWIFT code string

## Usage

``` r
is_bic(x)
```

## Arguments

- x:

  (`any`)  
  Object to check.

## Value

`TRUE` if `x` is a valid BIC/SWIFT code string, `FALSE` otherwise.
