# Check if an argument is an MD5 hash string

Check if an argument is an MD5 hash string

## Usage

``` r
is_md5(x)
```

## Arguments

- x:

  (`any`)  
  Object to check.

## Value

`TRUE` if `x` is a valid MD5 hash string, `FALSE` otherwise.

## Examples

``` r
is_md5("d41d8cd98f00b204e9800998ecf8427e")
#> [1] TRUE
```
