# Check if an argument is a valid CUSIP string

Validates CUSIP format including Luhn-variant check digit verification.

## Usage

``` r
is_cusip(x)
```

## Arguments

- x:

  (`any`)  
  Object to check.

## Value

`TRUE` if `x` is a valid CUSIP string, `FALSE` otherwise.
