# Check if an argument is an ISSN string

Validates ISSN format including check digit verification.

## Usage

``` r
is_issn(x)
```

## Arguments

- x:

  (`any`)  
  Object to check.

## Value

`TRUE` if `x` is a valid ISSN string, `FALSE` otherwise.
