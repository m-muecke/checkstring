# Check if an argument is a email address string

Check if an argument is a email address string

## Usage

``` r
is_email(x)
```

## Arguments

- x:

  (`any`)  
  Object to check.

## Value

`TRUE` if `x` is a valid email address string, `FALSE` otherwise.

## Examples

``` r
is_email("user@example.com")
#> [1] TRUE
```
