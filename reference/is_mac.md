# Check if an argument is a MAC address string

Check if an argument is a MAC address string

## Usage

``` r
is_mac(x)
```

## Arguments

- x:

  (`any`)  
  Object to check.

## Value

`TRUE` if `x` is a valid MAC address string, `FALSE` otherwise.

## Examples

``` r
is_mac("00:1B:44:11:3A:B7")
#> [1] TRUE
```
